About

This is a sample Flask application demonstrating the traditional MVC pattern:

  • Models - Task and Note classes with in-memory storage
  • Views - Jinja2 templates for rendering HTML
  • Controllers - Flask route handlers

Built to demonstrate the Light Table-style route view, which shows all related code for a route in one unified interface.

Features

  • Task management with completion tracking
  • Note taking with full-text display
  • Clean, responsive design
  • No database required (in-memory storage)
