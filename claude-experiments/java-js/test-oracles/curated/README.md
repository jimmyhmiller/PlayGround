# Curated Test Fixtures

This directory contains curated JavaScript source files from well-known open-source projects, used for parser testing and validation.

## Contents

| Directory | Source | Files | License |
|-----------|--------|-------|---------|
| `react/` | [React](https://react.dev/) | Development build | MIT |
| `framer-motion/` | [Framer Motion](https://www.framer.com/motion/) | Animation library | MIT |
| `motion-dom/` | [Motion DOM](https://motion.dev/) | Motion primitives | MIT |
| `watchpack/` | [Watchpack](https://github.com/webpack/watchpack) | File watcher | MIT |
| `eslint-plugin-unicorn/` | [eslint-plugin-unicorn](https://github.com/sindresorhus/eslint-plugin-unicorn) | ESLint rules | MIT |
| `typescript-tests/` | [TypeScript](https://www.typescriptlang.org/) | Parser test baselines | Apache-2.0 |
| `oxc-fixtures/` | [oxc](https://oxc-project.github.io/) | Parser test fixtures | MIT |

## File Versions

These files were extracted from npm packages as of December 2024:

- **React**: 19.x (experimental/development build)
- **Framer Motion**: 11.x
- **Motion DOM**: 11.x
- **Watchpack**: 2.x
- **eslint-plugin-unicorn**: 56.x
- **TypeScript**: Test baselines from TypeScript repo
- **oxc**: Test fixtures from oxc repo
