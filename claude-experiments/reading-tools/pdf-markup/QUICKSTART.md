# Quick Start Guide

## Open and Run

1. Open the project:
   ```bash
   open PDFMarkup.xcodeproj
   ```

2. In Xcode, select your run destination:
   - **For Mac**: Choose "My Mac (Designed for iPad)" 
   - **For iPad**: Choose an iPad simulator or your connected iPad

3. Click the Run button (▶) or press ⌘R

4. The app launches with a sample PDF already loaded!

## Start Highlighting

- **Switch colors**: Tap any colored circle at the top
- **Highlight**: Draw on the PDF with Apple Pencil, mouse, or trackpad
- **Navigate pages**: Use the arrow buttons at the bottom
- **Load your own PDF**: Click "Open PDF" button

## That's it!

No setup needed - the app is ready to use immediately.
